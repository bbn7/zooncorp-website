# Brand palette, typography, tokens

Use exact values in CSS.

* Primary green: `#2E6B4F`
* Accent forest: `#1E4F3A`
* Warm tan: `#C9A66B`
* Deep brown: `#5A3F2B`
* Off-white background: `#FAF8F4`
* Neutral text: `#222222`
* UI radius: `12px`
* Elevation shadow: `0 6px 20px rgba(34,34,34,0.08)`

Fonts (web): Inter for UI, Playfair Display for headings. Use Google Fonts:
`https://fonts.googleapis.com/css2?family=Inter:wght@300;400;600;700&family=Playfair+Display:wght@400;700&display=swap`

Design principles: strong left-aligned content grid, large readable type, rich hero with split image/text, card-based services, concise global-standard legal pages.

# SEO package (meta + structured data) — include in each page in HEAD

Add this JSON-LD snippet in `<head>` of main pages:

```html
<script type="application/ld+json">
{
  "@context":"https://schema.org",
  "@type":"Organization",
  "name":"ZOON CORP Pvt Ltd",
  "url":"https://www.zooncorp.com",
  "logo":"https://www.zooncorp.com/logo/zoon-logo.svg",
  "contactPoint":[{"@type":"ContactPoint","telephone":"+91 1800 555 5555","contactType":"customer service","areaServed":"GLOBAL","availableLanguage":["English"]}],
  "sameAs":[]
}
</script>
```

Top 5 SEO recommendations (implement now):

1. Add canonical link to each page: `<link rel="canonical" href="https://www.zooncorp.com/yourpage">`.
2. Ensure hero image has `alt` describing content and `width/height` where possible.
3. Host images in `assets/img/` with optimized WebP versions; serve WebP via `<picture>`.
4. Add hreflang only if multiple locales.
5. Request indexing via Google Search Console and submit `sitemap.xml`.

# Accessibility & performance checklist (must-do)

* Add `lang` attributes and skip links if content long.
* Run Lighthouse: target performance > 90, accessibility > 90.
* Compress images, serve WebP/AVIF.
* Add `width`/`height` for images for CLS reduction.
* Add `rel="preconnect"` for Google Fonts and `font-display:swap`.
* Implement basic caching headers (via Cloudflare Workers or hosting).

---

# Legal and compliance (immediate edits)

* Update Privacy Policy last-updated date and add DPO contact if collecting personal data.
* For exports: add compliance disclaimers and export license contact in `about.html` or `services.html`.
* Include cookie banner if tracking (even basic analytics).
